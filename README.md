Node.js examples
=====

This repository contains example applications in Javascript for the Node.js runtime.


This repository shows the following features of Node.js:
* HTTP Server
* Async/Await
* Readfile
* Streams
* Set timeout
* Async promises
* Async Axios requests
* JSON parse
* Express framework
